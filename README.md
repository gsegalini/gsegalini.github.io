# gsegalini.github.io

Personal academic website for Giulio Segalini, built with Jekyll and the
al-folio theme.

## Local development

```bash
docker compose up --build
```

The site runs at <http://localhost:8080>.

## Content to customize

- `_pages/about.md`: homepage bio and profile details
- `_bibliography/papers.bib`: publications and preprints
- `_data/cv.yml`: rendered CV content
- `_data/socials.yml`: public links and contact channels
- `_projects/`: research and software project pages
- `assets/img/`: profile and project images
- `assets/pdf/`: CV and publication PDFs
