Jamie is a local-first voice assistant Discord bot. It joins a voice channel, listens for a wake word, transcribes speech locally, uses an LLM to extract intent, and dispatches commands through an extensible plugin system.

It supports music-bot voice control, per-server Q&A memory, optional text-to-speech responses, and local model inference by default, with optional cloud LLM backends.
